# 系谱图遗传方式判定

## 钩子

同学们，我描述一个你们都经历过的场景。拿到一道系谱题，看了一眼图——"双亲正常、孩子患病"。心里想：隐性。提笔就写"常染色体隐性遗传"。卷子发下来，扣了一半的分。

为什么？因为你跳了一步。"无中生有"只能告诉你这是隐性——它不能告诉你这个基因在常染色体上还是在X染色体上。跳过这一步，结论就是猜的。

今天这节课，我把系谱判型变成一个你绝对不会跳步的固定程序。学完之后，你看到系谱图，脑子里不是一团模糊的感觉，而是一条清晰的流水线。

## 信号

在题目里，什么信号告诉你"这是一道系谱判型题"？

第一个信号，**系谱图本身**。方框圆圈，横线竖线，黑色白色。看到这个格式，就是系谱题。不需要别的信号——图本身就是最大的信号。

第二个信号，**"遗传方式"或"最可能的遗传方式"**。题目直接问"该病的遗传方式是"。这是赤裸裸的判型题。如果加了"最可能的"四个字——说明信息不够全，你要选一个所有已知信息都能兼容的选项。

第三个信号，**"排除"问法**。"下列哪种遗传方式可以被排除""下列哪种情况不可能"。这类题不走完整判型——只走排除。你要逆向思考：哪种遗传方式和系谱中某个个体的表型矛盾。

第四个信号，**叠加信息**。系谱图之外还有文字条件——"父亲不携带致病基因""Ⅱ-2是纯合子""该基因位于X染色体上"。这些条件是你判型的硬约束。先读图再看文字条件，顺序不能反。

## 判断

系谱判型的判断，是一个**两步排除法**。

**第一步：判显隐。** 在系谱里找"无中生有"或"有中生无"。双亲正常、生出患病孩子——无中生有，隐性。双亲患病、生出正常孩子——有中生无，显性。两种都没找到——看系谱特征（代代出现倾向于显性，隔代出现倾向于隐性），但注明"推测"。

**第二步：判常染色体还是伴X。** 找一个关键个体——一个能用它的存在来一票否决一整类遗传方式的人。

记住三句一票否决口诀：
- 患病女性 + 父亲正常 → 一票否决 X 隐。因为X隐女患者两条X都带致病基因，一条必须来自父亲，父亲必患病。
- 患病男性 + 母亲正常 → 一票否决 X 显。因为X显男患者的X只有一个来源——母亲。母亲正常，儿子不可能患病。
- 患病父亲 + 正常女儿 → 一票否决 X 显。因为X显父亲的致病X必传每一个女儿。只要有一个正常女儿，就不是X显。

这两步走完，结论才可靠。在草稿纸上分两行写：第一行——隐/显。第二行——常/X待判。两行写满之前，不许写结论。

## 框架

所有系谱判型题只有四种可能：常隐、常显、X隐、X显。

这四种的核心区别在于：**致病基因在哪条染色体上，一条还是两条能致病。**

常染色体上：男女患病概率相同。隐性——两条都带致病基因才患病。显性——一条带致病基因就患病。

X染色体上：男女患病概率不同。隐性——男性只需一条X带致病基因就患病（因为他只有一条X），女性需要两条。显性——女性一条带致病基因就患病，男性带致病基因就患病但通常更严重。

搞定这四种，任何系谱都脱不出这个框。判显隐锁定两类（隐/显），判位置锁定终极答案（常/X）。

## 拆题

### 第一道母题：常染色体隐性——镰状细胞贫血

看一个家系。1号男性正常，2号女性正常。他们生了一个女儿——4号，患病。

第一步，判显隐。双亲正常，生出患病孩子——典型的无中生有。致病基因为隐性。

第二步，判常还是伴X。找系谱里的患病女性，看她的父亲。4号是女患者，她父亲是1号。1号正常，不患病。现在问自己：如果这个病是X隐，4号女患者基因型必须是XaXa——其中一条Xa必须来自父亲。那父亲应该是XaY——患病。但1号正常。排除X隐，锁定常染色体隐性。

两步走完，结论才是可靠的。

第三步，推基因型。假设致病基因为a，正常基因为A。4号aa。1号和2号各给了她一个a——都是杂合子Aa。

第四步，算概率。父母Aa×Aa，后代AA:Aa:aa=1:2:1。如果题目说"8号表型正常"，先划掉aa——8号可能是AA或Aa，比例1:2。8号是携带者Aa的概率是2/3，不是1/2。如果你忘了调分母，用1/2算，差将近一倍。

### 第二道母题：X染色体隐性——卡尔曼综合征

小红正常，丈夫正常。他们生了一个儿子患病。小红的弟弟也患病。小红父母都正常，题干加了一句："父亲不携带致病基因"。

第一步，无中生有——隐性。

第二步，用"父亲不携带"做排除。先试常隐：弟弟aa，父母必须各给一个a，都是Aa——但父亲"不携带"，应该是AA，给不出a。常隐排除。再试X隐：弟弟XaY，Xa只能来自母亲。母亲XAXa——携带者，表型正常。父亲给弟弟Y染色体，父亲XAY正常且"不携带"——完全吻合。结论：X染色体隐性遗传。

你发现没有？两道题第一步完全一样。但第二步——一个是靠"患病女性+正常父亲"排除X隐，一个是靠"父亲不携带"排除常隐。路径不同，逻辑一样：找一个能一票否决的关键信息。

### 第三道母题：常染色体显性——EOFAD

早发性阿尔茨海默病家系。代代都有患者。Ⅱ-1"不携带致病基因"。Ⅱ-2女患者，和Ⅱ-1生了一个正常孩子Ⅲ-1。

第一步，没有典型无中生有或有中生无。但代代出现——显性特征。且Ⅱ-1不携带。

假设显性：Ⅱ-2杂合子Aa，Ⅱ-1不携带（aa），Aa×aa→一半患病一半正常。Ⅲ-1正常——说得通。配合代代出现，显性解释力明显更强。

第二步，判常还是伴X。找患病男性，看女儿。系谱验证后排除X显，锁定常染色体显性。

关键工具："不携带致病基因"在显性判定中特别有用。正常人不携带+患者，生出患病孩子——致病基因只来自患者，患者能把致病基因传下去→显性。

### 第四道母题：X染色体显性——雷特综合征

父亲患病，"只具致病基因"——没有正常等位基因。生了一个儿子正常，一个女儿患病。

第一步，患病父亲生出正常儿子——有中生无，显性。

第二步，父亲只具致病基因。如果是常显——父亲AA，跟正常aa结婚，后代全是Aa全部患病。但儿子正常——排除常显。如果是X显——父亲XAY。XA传给每个女儿——女儿全患病。Y给儿子——儿子从母亲获得X，母亲正常则儿子正常。全部吻合。结论：X染色体显性。

核心区分逻辑：父亲只含致病基因时——女儿全患→X显。儿女都患→常显。因为X染色体只从父亲传给女儿。

## 陷阱

第一个坑：无中生有直接判常隐。看见双亲正常孩子患病，自动补了"常染色体"四个字。强制在草稿纸上分两行写：第一行——隐性。第二行——常/X待判。两行写完才许写结论。

第二个坑：条件概率忘调分母。Aa×Aa后代AA:Aa:aa=1:2:1。已知表型正常——不可能是aa，正常个体中携带者的概率是2/3，不是1/2。

第三个坑：用男女比例判断伴X。五人家系的男女比例是完全随机的。判断伴X只有一个硬标准——关键个体的一票否决。不要看比例，去找那个能一票否决的人。

第四个坑：电泳条带上下搞反。片段越小跑得越远——位置越靠下。上面是大片段，下面是小片段。做系谱+电泳联合题时，从系谱表型反推条带含义，不要先猜条带再套系谱。

第五个坑：近亲结婚——问你禁止近亲结婚是针对什么遗传病。很多同学写显性。大错。近亲结婚真正推高的是隐性遗传病的风险——因为近亲携带相同隐性致病基因的概率远高于随机人群。一句话记住：近亲结婚，隐性遭殃。

## 收束

这节课走了四道母题——常隐、常显、X隐、X显。每道题都是同一条流水线：判显隐→判位置→推基因型→算概率。这个程序你不需要理解，你只需要执行。走多了就变成肌肉记忆。

高考里的系谱题通常是八到十二分大题的第一问。第一问判稳了，后面的电泳、PCR、突变、减数分裂叠加才立得住。先清主场，再处理叠加——顺序不能反。

课后按四步练。第一步练判显隐：EC-143(1)、EC-135(1)、EC-188(2)、EC-192(1)——四道分别对应常隐、常显、X隐、X显。第二步练一票否决：EC-162(6)和EC-129(2)。第三步练条件概率：EC-162(8)。第四步综合实战：EC-137的(1)(2)(7)。

系谱判型，永远先判显隐再判位置，永远找那个能一票否决的关键个体。
