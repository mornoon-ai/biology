# 细胞结构变化与功能适应

## 钩子

同学们，你在生物课上反复听过一句话——结构决定功能。这四个字反过来就是今天全部的考题：**结构变了，功能就一定变。**

高考不会让你默写这句话。它会给你一个具体的突变、一个形态变化、一个蛋白异常，让你往下推出功能后果。这节课我们练的就是这个推演能力。

## 信号

题目里什么信号告诉你"出题人在考结构→功能推理"？

第一个信号，**"基因突变→蛋白异常"链**。题干给出"XX基因突变导致XX蛋白结构改变"，然后问"这会导致什么功能异常"。这是在考你：蛋白结构变了→蛋白不能正常工作→某细胞功能丧失→某生理过程受阻。

第二个信号，**细胞形态描述**。"细胞皱缩""微绒毛减少""线粒体肿胀""膜内陷""囊泡堆积"——这些是形态学术语。看到形态变化，要立刻反推：这个结构正常时是干什么的？

第三个信号，**"阻断"或"缺失"效应**。题目说"敲除X基因后，细胞内的Y物质堆积"——说明X基因产物是Y物质运输或加工所必需的。堆积处是断点的上一站。

第四个信号，**功能后果描述**。"不育""神经退行""代谢紊乱"——这些是最终表现。题目要你从基因突变一路推到最终表现。

## 判断

拿到结构功能推理题，不要一开始就想答完整。先做结构拆解。

**第一步：识别被改变的结构。** 题干给的是什么？一个基因？一个蛋白？一个细胞器？一条运输通路？锁定目标。

**第二步：锁定这个结构正常时的功能。** TUB基因编码微管蛋白→微管是囊泡运输的轨道。半胱天冬酶→切割细胞骨架蛋白。你不需要知道所有细节，但必须知道这个结构在细胞里的"日常工作"是什么。

**第三步：沿因果链推下去。** 从结构改变出发，经过"日常工作被阻断"，推到"细胞层面后果"，再推到"组织/个体层面表现"。每一步之间必须有明确的因果连接词——"导致""引起""因而""因此"——不能跳步。

## 框架

结构功能推理题的通用框架：**五步推演链。**

什么结构 → 怎么变的 → 这个结构正常时干什么 → 变了之后干不了什么 → 最终表现是什么

把这五个问题依次回答，任何一道结构功能题都能拿满分。五步之间不能跳——每个箭头都要有因果连接，每个后果都要来自前一步的变化。

## 拆题

### 第一道母题：TUB基因突变与囊泡运输瘫痪

TUB基因编码微管蛋白。微管是细胞骨架的一部分，在细胞里像铁轨一样铺开。驱动蛋白是火车头，拖着囊泡在微管轨道上定向移动。

如果TUB基因突变，微管蛋白结构异常——铁轨坏了。后果不是一节车厢出轨，而是整条运输线瘫痪。内质网到高尔基体的囊泡过不去，高尔基体到细胞膜的囊泡也过不去。

具体到卵母细胞：微管运输瘫痪后，卵母细胞减数分裂需要的蛋白质和信号分子运不进来。缺少这些物质，减数分裂无法正常进行——卵母细胞成熟被阻滞。最终表现是不育。

考题会这样考：题干给你"TUB基因突变→微管异常"，问你"卵母细胞不能成熟的原因是什么"。你要答出完整链：TUB基因突变→微管蛋白结构异常→微管组装缺陷→驱动蛋白无法沿微管行走→囊泡运输中断→减数分裂所需物质无法到达→减数分裂阻滞→卵母细胞不能成熟→不育。每一步都不能跳。

### 第二道母题：凋亡小体——有秩序的自毁

细胞凋亡和细胞坏死是两回事。坏死是被动的——细胞被物理或化学损伤撑破，内容物流出，引发炎症。凋亡是主动的——细胞接收凋亡信号后，有步骤地把自己拆掉。

凋亡的第一步：细胞皱缩。半胱天冬酶切断细胞骨架蛋白，细胞失去支撑，体积缩小。第二步：膜内陷。细胞膜像气球被手指按压，形成一个个凹陷。第三步：凹陷加深、断裂——形成被膜包裹的小泡，里面装着细胞碎片。这就是凋亡小体。

凋亡小体的命运：被邻近的巨噬细胞识别并吞噬。在巨噬细胞的溶酶体里被降解。整个过程干干净净，不引发炎症——这是凋亡和坏死最大的区别。

考题经常问："凋亡过程体现了生物膜的什么特性？"答案是流动性——膜内陷和断裂成小泡，需要膜分子的流动和重组。"溶酶体在凋亡中的作用是什么？"——降解吞噬进来的凋亡小体。

用五步推演链复盘：细胞骨架（结构）→被半胱天冬酶切断（怎么变的）→骨架维持细胞形态（正常功能）→形态维持不了→细胞皱缩、膜内陷（表现）。

## 陷阱

第一个：把凋亡和坏死混为一谈。题目说"细胞肿胀破裂，内容物外泄引发炎症"——坏死。题目说"细胞皱缩，膜内陷形成凋亡小体，被邻近细胞吞噬"——凋亡。核心区分：凋亡是主动有序的，坏死是被动无序的。凋亡形成凋亡小体，坏死直接破裂。凋亡不引发炎症，坏死引发炎症。

第二个：答题时跳步，不建立完整因果链。从"微管坏了"直接跳到"不育"，中间略过了囊泡运输中断和减数分裂阻滞。少一个环节就少一分。完整链至少要有五个环节，每个环节一个箭头。

## 收束

结构功能推理题就一条公式：什么结构→怎么变→正常做什么→变后不能做什么→最终什么后果。五个问题依次答，不跳步。把每个箭头前面的"导致"写在草稿纸上，练成肌肉记忆。

课后做两道题。EC-191 的第(3)(4)(5)问——TUB突变和囊泡运输，练微管到运输瘫痪的完整因果链。EC-494 的第(3)(4)问——凋亡小体形成与溶酶体降解，练凋亡的形态变化和膜流动性的体现。
