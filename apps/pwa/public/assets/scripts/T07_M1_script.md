# 食物链与营养级分析

## 钩子

同学们，生态学大题的第一问通常是让你写食物链、判断营养级。这看起来简单，但有一个坑很多人跳了无数次——**同一个生物可以在不同的食物链里处于不同的营养级。**

## 信号

题目里什么信号告诉你"这是食物链与营养级题"？

第一个信号，**"食物链""食物网""营养级"**。直接出现这些术语——进入生态学营养结构框架。

第二个信号，**物种关系描述**。"A以B为食""C捕食D""土壤中XX数量增多导致YY数量减少"——捕食关系串联成链。

第三个信号，**"级联效应""顶级捕食者"**。一个物种数量变了→整个链波动→问连锁反应。顺箭头方向推。

第四个信号，**"杂食动物"**。看到"既吃...又吃..."——同一个生物在多条链中占据不同营养级。

## 判断

拿到食物链题，两步判定。

**第一步：数营养级。** 生产者永远是第一营养级（植物、藻类、蓝藻）。吃生产者的→第二营养级。吃第二营养级的→第三营养级。以此类推。

**第二步：杂食动物的多链判定。** 海獭吃螃蟹（螃蟹吃鳗草→海獭第三级），海獭吃贝类（贝类滤食浮游植物→海獭也是第三级）。人在不同链中可以是第二级（吃粮食）也可以是第三级（吃牛肉）。

## 框架

食物链的底层规则：箭头方向=能量流动方向=被捕食→捕食者。营养级从生产者开始数。每条链独立计数，同一生物在不同链中营养级可不同。

级联效应：一个物种数量变化→沿食物链传播。捕食者减少→猎物增多→猎物的食物减少。顺箭头方向推，不跳步。

## 拆题

考题给生态系统物种关系表。先画出所有食物链→每条链标营养级→找出处于最高营养级的生物（顶级捕食者）。

考题："如果海獭消失，鳗草会怎样？"海獭减少→螃蟹增多→鳗草被大量啃食→鳗草减少。顺箭头推三步：海獭↓→螃蟹↑→鳗草↓。

## 陷阱

第一个：把杂食动物固定在一个营养级。看到"人吃粮食"，就写人是第二营养级。但人也吃牛肉→第三营养级。杂食动物没有固定营养级。

第二个：营养级从消费者开始数。生产者是第一营养级，不能跳过。

## 收束

营养级从生产者开始数。杂食动物多链多级。级联效应顺箭头推——捕食者减→猎物增→再下级减。
