# 基因编辑与RNA干扰

## 钩子

同学们，如果基因组是一本很厚的书，我今天教你两件事。第一件——怎么在这本书里找到一个特定的字，然后把它改掉。第二件——不改书，但是某些页在印刷之前被拦下来，不让它们变成成品。

改书的是基因编辑。拦印刷的是RNA干扰。两件事原理完全不同，但高考喜欢放在一起考——因为它们都是"操控基因表达"的手段。

## 信号

在题目里，怎么看出一道题考的是哪种分子技术？

看到 **"Cas9""sgRNA""PAM""NHEJ""HDR""双链断裂"** ——基因编辑。出题人在问你：DNA层面的切割和修复。

看到 **"Dicer""RISC""miRNA""siRNA""mRNA降解""转录后调控"** ——RNA干扰。出题人在问你：mRNA被拦截了，蛋白质造不出来。

看到 **"甲基化""组蛋白修饰""印记""序列不变"** ——表观遗传。出题人在问你：序列没变，表达变了。

三道门，三个钥匙词群。题目里出现了哪一堆词，就进入哪个框架。

## 判断

拿到分子技术题，先别急着答。用三个词做快筛。

**Cas9？→ 基因编辑。** 问的是DNA怎么被切开、怎么修复。

**Dicer / RISC？→ RNA干扰。** 问的是mRNA怎么被标记和销毁。

**甲基化 / 印记？→ 表观遗传。** 问的是序列不变时怎么调控表达。

注意——这三个方向经常被出题人混在一道大题里。第(1)问考CRISPR原理，第(3)问可能让你判断"RNA干扰属于表观遗传吗"（答案：不属于，因为RNA干扰不改变染色质状态）。一题考多个方向，你必须能快速切换框架。

## 框架

两个系统，一张图记住。

**基因编辑 = 切 + 修。** Cas9是剪刀，sgRNA是导航。sgRNA找到目标序列，PAM确认"这里是合法切割位点"，Cas9切开双链。切完后，细胞自己修——没模板就走NHEJ（随机删几个碱基，基因废了），有模板就走HDR（把你的序列嵌进去）。

**RNA干扰 = 拦。** Dicer把长RNA切成小片段，RISC复合物带着小片段在细胞质里巡逻。碰到mRNA跟小片段互补配对——要么切掉mRNA，要么卡住不让翻译。DNA没动，但蛋白质造不出来。

核心区别一句话：基因编辑动的是DNA，是永久的。RNA干扰拦的是mRNA，是可逆的。

## 拆题

### 第一道母题：CRISPR敲除基因

题目给你一个实验场景：研究者设计了针对某个致病基因的sgRNA，用Cas9处理细胞后，检测到这个基因的蛋白表达完全消失了。问：机制是什么？

你的答案要分三步写。第一步：sgRNA通过碱基互补配对识别目标基因。第二步：Cas9在PAM位点上游切断DNA双链。第三步：细胞通过NHEJ修复断裂——随机删除或插入碱基→移码突变→基因失活。

题目如果追问："这个基因还能被修复吗？"——不能轻松修复，因为NHEJ带来的随机缺失是不可预测的，留下的是缺口和移码。

题目如果变一下："研究者同时提供了一段正常的基因片段，处理后的细胞表达了正常蛋白。"那修复方式就是HDR——模板把正确的序列补回去了。

### 第二道母题：miRNA调控实验

题目给你一组数据：导入某microRNA后，基因X的mRNA水平不变，但基因X的蛋白质表达下降70%。问机制。

抓两个关键。mRNA水平不变——说明基因转录正常，DNA没被动。蛋白质下降——说明翻译被阻断了或者mRNA被切了但还没完全降解。

答案是RNA干扰：miRNA→Dicer加工→装进RISC→与基因X的mRNA 3'UTR不完全配对→翻译抑制。注意"不完全配对"——miRNA和靶mRNA不需要完全互补，部分配对就能抑制翻译。这和siRNA不同——siRNA通常完全配对，直接导致mRNA切割。

知识点：miRNA主要靠翻译抑制，siRNA主要靠mRNA切割。但两者都经过Dicer→RISC通路。

## 陷阱

第一个陷阱：把Cas9当成全能选手。Cas9是剪刀，不是胶水，不是模板。剪完之后怎么修，Cas9管不了。题目问你"Cas9的作用是什么"——答"识别特定DNA序列并在PAM位点附近切割DNA双链"，不要多答"敲除"或"敲入"。

第二个陷阱：以为miRNA和siRNA是完全不同的东西。它们来源不同——miRNA是内源的，siRNA通常是外源的。但进入细胞后的通路几乎一样：都经过Dicer加工，都被装进RISC，都通过碱基配对来抑制翻译或降解mRNA。说它们"完全不同"是错的，说它们"完全一样"也是错的。同路不同源。做题时看题目问"来源"还是"通路"——两个答案不一样。

## 收束

这节课两个系统，一个切DNA，一个拦mRNA。切DNA的记住：sgRNA找→PAM确认→Cas9切→细胞修。拦mRNA的记住：Dicer切→RISC装→碱基配对被拦截。

课后做两道题。第一道，做 EC-007 的第(1)(2)问——CRISPR基础，练sgRNA与PAM的识别和NHEJ/HDR的区别。第二道，做 EC-202 的第(2)(3)(4)问——miRNA通路，练Drosha到RISC的完整加工链和miRNA/siRNA的区分。
