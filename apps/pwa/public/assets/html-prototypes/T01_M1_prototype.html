两条子专题
子专题 1
构象异常 → 功能丧失 / 信号失调
单个氨基酸改变（HbS: Glu→Val；CFTR: 氨基酸缺失）→ 空间结构改变 → 正常功能（运输/信号）丧失或异常亢进 → 疾病表现
子专题 2
错误折叠聚集 → 自噬受阻 → 细胞死亡
错误折叠蛋白形成寡聚体/沉积（Aβ/αSyn）→ 干扰自噬清除机制（BAX/Mcl-1通路）→ 损伤细胞器堆积 → 神经元死亡
高频易错
氨基酸种类改变 ≠ 氨基酸数目改变。HbS是第6位Glu→Val（替换），数目不变；CFTR是氨基酸缺失，数目减少。
子专题1中Pyrin蛋白（FMF）不是"功能丧失"，而是"功能异常亢进"——对刺激过度敏感，触发正反馈放大IL-1。
Mcl-1位于线粒体外膜，不在细胞质基质。正常状态下Mcl-1与LC3A结合促进自噬；患者体内BAX竞争性与Mcl-1结合，阻断自噬。
αSyn聚集过程存在正反馈：寡聚体→促进更多αSyn磷酸化→更多寡聚体→恶性循环。
HbS疏水突起与疏水口袋结合（不是与另一个疏水突起结合），形成多聚体纤维。
子专题1 母题 镰状细胞贫血·HbA→HbS·疏水突起聚合·红细胞变形 ★ 10/10
EC-009 · (1)(2)(3) · 2026-05-18
核心因果链：β链第6位 Glu（亲水）→ Val（疏水）→ β亚基产生疏水突起 → 疏水突起与另一HbS的疏水口袋结合 → 多聚体 → 不溶性纤维 → 红细胞镰刀变形 → 运氧能力下降
一级结构→功能的完整链
一个氨基酸替换→ 疏水性改变→ 空间结构改变→ 异常聚合→ 细胞形态变化→ 功能丧失
计算考点：脱水缩合水分子数 = 氨基酸总数 − 肽链条数 = (141×2+146×2)−4 = 570；游离羧基至少4个（每条链末端各1个）。
子专题2 母题 阿尔茨海默症·Aβ沉积·BAX阻遏Mcl-1·线粒体自噬受阻·UMI-77治疗 ★ 9/10
EC-486 · (4)~(8) · 2026-05-24
核心因果链：Aβ错误折叠沉积 → BAX与Mcl-1竞争性结合 → LC3A无法与Mcl-1结合 → 线粒体自噬受阻 → 损伤线粒体堆积 → 神经元退行
分子互作关键
正常：Mcl-1 + LC3A → 自噬囊泡形成 → 清除损伤线粒体
患者：BAX竞争与Mcl-1结合 → LC3A被排斥 → 自噬受阻
UMI-77：与Mcl-1结合 → 替换BAX → 恢复LC3A结合 → 自噬恢复
Mcl-1位于线粒体外膜（②），基本骨架是磷脂双分子层。BAX蛋白在核糖体（③）上合成。Aβ前体APP加工异常涉及内质网③、运输囊泡④、高尔基体⑤。
