核心概念 · 四类激素调控
生长素（IAA）两重性
低浓度促进生长，高浓度抑制生长；根<芽<茎对生长素的敏感性（根最敏感）；极性运输（形态学上端→下端，主动运输）；横向运输在弯曲部位。
赤霉素（GA）
促进细胞伸长（节间）→茎秆增高；促进种子萌发和果实发育；外施GA→矮生植物变高（可替代正常高度基因功能）；赤霉素缺乏→矮生表型。
脱落酸（ABA）
抑制生长、促进脱落和种子休眠；干旱→ABA↑→气孔关闭→减少水分散失；种子成熟时ABA积累→休眠；与赤霉素拮抗（GA促萌发、ABA抑萌发）。
乙烯
气体激素；促进果实成熟（催熟）；抑制细胞伸长（高浓度）；乙烯合成在果实成熟过程中正反馈加速；外施乙烯或与成熟果实同放→催熟未熟果实。
高频陷阱（5 个）
生长素两重性的"高浓度抑制"是相对的——对茎来说是促进的浓度，对根来说可能已经是高浓度抑制了；不能脱离靶器官单独判断高低浓度效果
极性运输是主动运输（需要载体蛋白和能量），方向固定为形态学上端→下端；而横向运输（侧方光照引起）是被动的，属于横向不均等分布，两者机制不同
顶端优势中，顶芽产生的生长素向下运输积累到侧芽处→侧芽处生长素浓度过高→侧芽受抑制；去除顶芽后侧芽处浓度降低→侧芽萌发。不能说顶芽"分泌抑制激素"
植物激素不是蛋白质（脱落酸是萜类化合物，生长素是吲哚乙酸，乙烯是气体），不能用蛋白质变性或高温破坏来消除其活性——这是考查化学本质的常见陷阱
赤霉素和生长素都能促进果实发育（无子果实形成），但机制不同：赤霉素主要促进细胞体积增大；生长素促进子房发育；两者协同但不能互相替代所有功能
生长素 生长素两重性——浓度-效果曲线的三个关键点解读与顶端优势 10/10
EC-480 · 子问题 (1)(2)(3)
核心信号：题目出现"生长素浓度-效果坐标图"或询问"高浓度生长素对某器官的效果"，或分析顶端优势的原理。需辨析根、芽、茎对生长素敏感性的差异及两重性表现。
浓度-效果曲线三个关键坐标点
促进最大点（最适浓度）：效果曲线顶点；根<芽<茎最适浓度依次升高
零效果点（既不促进也不抑制）：曲线与X轴交点；高于此点→开始抑制
最高浓度（强烈抑制）：远高于最适浓度→与最大抑制效果对应
顶端优势机制
顶芽合成IAA→极性运输到侧芽→侧芽IAA浓度高（超最适）→侧芽受抑制 ✓
摘除顶芽→侧芽IAA浓度降低到促进范围→侧芽萌发→分枝增多

根对生长素最敏感（最适浓度约10⁻¹⁰mol/L），茎最不敏感（约10⁻⁶mol/L）。根的弯曲生长：向光侧IAA少→促进较弱；背光侧IAA多→促进较强→根弯向光侧（向地弯曲）。

赤霉素 赤霉素促进节间伸长与矮生植物突变体实验分析 10/10
EC-485 · 子问题 (1)(2)
核心信号：题目提供"矮生植物突变体"或"外施赤霉素后植物变高"的实验，要求判断突变的原因（赤霉素合成缺陷 vs 受体缺陷）以及赤霉素的作用靶点。
赤霉素作用机制
赤霉素合成→与细胞受体结合→促进细胞纵向伸长（节间）→茎秆增高 ✓
赤霉素还促进：种子萌发（解除休眠）+果实发育（无子果实）
矮生突变体实验判断
外施赤霉素→植物变高→突变为赤霉素合成缺陷（受体正常）✓
外施赤霉素→植物仍矮→突变为赤霉素受体缺陷（无法响应激素）

赤霉素与生长素协同促进茎的生长：生长素促进细胞分裂（分裂素），赤霉素促进细胞伸长，两者协同作用远大于单独作用。这是激素协同作用的典型考点。另：外施赤霉素处理大麦种子→α-淀粉酶分泌增加→淀粉水解→为萌发提供能量，是萌发激素调控的重要机制。

脱落酸 ABA促进种子休眠与干旱胁迫下气孔调节的信号通路分析 9/10
EC-490 · 子问题 (1)(2)(3)
核心信号：题目提供"干旱→ABA→气孔关闭"信号链，或要求分析种子休眠/萌发中ABA与GA的拮抗关系，或提供ABA受体突变体实验数据。
ABA双重调控功能
种子成熟期：ABA积累→抑制萌发→种子进入休眠
种子萌发期：GA↑拮抗ABA↓→解除休眠→萌发启动 ✓
干旱胁迫→叶片ABA合成↑→保卫细胞K⁺外流→气孔关闭→减少蒸腾 ✓
ABA受体突变体分析（实验常考）
ABA受体功能缺失突变体→无法感受ABA信号→干旱下气孔不关闭→植物萎蔫 ✓
ABA受体功能增益突变体→即使正常水分条件也关闭气孔→生长减慢

ABA的化学本质：倍半萜烯（萜类化合物），不是蛋白质。合成部位：叶肉细胞（干旱时大量合成）。运输方式：韧皮部运输（可长距离）。⚠️ABA与乙烯协同促进叶片/果实脱落；与GA拮抗（GA促萌发，ABA抑萌发）。

乙烯 乙烯催熟果实的正反馈机制及激素间协同与拮抗关系综合分析 9/10
EC-495 · 子问题 (1)(2)(4)
核心信号：题目要求分析"乙烯催熟果实"的机制，或在综合激素题中要求判断"协同/拮抗"关系，或给出不同激素处理实验要求解释结果。
乙烯催熟正反馈
果实发育后期→乙烯合成↑→促进果实成熟（色、香、软）
成熟过程→进一步刺激乙烯合成↑（正反馈）→加速成熟进程 ✓
激素协同与拮抗总结
协同（加强效果）：生长素+赤霉素→促茎生长，ABA+乙烯→促脱落 ✓
拮抗（相互对抗）：GA促萌发 ↔ ABA抑萌发，生长素促茎生长 ↔ 乙烯（高浓度）抑茎伸长

乙烯是气体激素，化学本质是不饱和烃（CH₂=CH₂），非蛋白质。乙烯效应还包括：促进叶片/果实脱落（与ABA协同），抑制茎的纵向伸长（高浓度下茎横向增粗）。外施乙烯释放剂（乙烯利）→催熟未熟果实，是农业中常用的应用方式。
