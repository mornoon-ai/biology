核心概念 · 四个分析维度
净光合 vs 总光合
表观（净）光合速率 = 总（真正）光合速率 − 呼吸速率；坐标图中测得的通常是净光合（O₂释放量或CO₂吸收量）
光补偿点与光饱和点
光补偿点：净光合=0，光合=呼吸；光饱和点：继续增光，光合速率不再升高；超饱和点后，CO₂/温度成为限制因素
三大限制因子
光照强度（光反应限速）、CO₂浓度（暗反应C3合成限速）、温度（酶活性限速）——同一时刻只有一个是最直接的限制因子
有机物积累量计算
白天积累 = 净光合 × 光照时数；夜晚消耗 = 呼吸速率 × 暗处时数；一天净积累 = 白天积累 − 夜晚消耗
高频陷阱（5 个）
坐标图中"O₂释放量"是净光合速率，不是总光合速率——要求总光合时必须加上呼吸速率（暗处耗O₂量）
光饱和点之后继续增加光照，光合速率不再增加——此时限制因子变成CO₂浓度或温度，不是光照
温度影响光合速率的两个方向：温度升高→酶活性↑（促进）；超过最适温度→酶失活（抑制）——呼吸速率也同时随温度变化
计算一天有机物净积累时，别忘了夜间呼吸消耗——只用白天净光合量会高估积累量
光补偿点处净光合=0，但植物仍在进行光合和呼吸——不能说"植物不进行光合作用"；有机物总量不变但两者速率相等
曲线解读 光照强度-光合速率坐标图的关键点定义与读取 10/10
EC-410 · 子问题 (1)(2)
核心信号：给出"光照强度（x轴）-O₂释放量（y轴）"坐标图，要求识别光补偿点、光饱和点，并读出暗呼吸速率和最大光合速率。标准母题：三点两线解读法。
坐标图三关键点
纵轴截距（负值）→暗呼吸速率（黑暗中O₂消耗量）
曲线与x轴交点→光补偿点（净光合=0，光合=呼吸）✓
曲线趋于平缓处→光饱和点（最大净光合速率）✓
总光合速率计算
总光合=净光合（图上值）+|暗呼吸速率| ✓

⚠️ 坐标图中测到的O₂释放量 = 净光合速率（总光合已扣除呼吸）；总光合 = 净光合 + 呼吸。这是最常出错的换算关系。

限制因子 光饱和点后限制光合速率继续增大的因子判断 10/10
EC-415 · 子问题 (2)(3)
核心信号：题目给出"光照强度达到光饱和点后，继续增加光照，光合速率不再升高"，要求判断此时限制光合速率增加的因素，并说明如何验证。
限制因子判断逻辑
光照充足（光反应不限速）→限制因子转向暗反应
增加CO₂浓度→光合速率升高 → CO₂是限制因子 ✓
适当升温→光合速率升高 → 温度是限制因子 ✓

验证方法（单一变量原则）：在光饱和点光照强度下，分别改变CO₂浓度或温度，观察光合速率是否升高。若升高则该因素是当前限制因子。

净/总光合计算 一昼夜有机物净积累量的计算（含白昼净光合与夜间呼吸） 10/10
EC-418 · 子问题 (3)(4)
核心信号：题目给出光照时数、净光合速率（或O₂释放量）、黑暗时数、呼吸速率，要求计算一昼夜有机物净积累量，或判断植物能否正常生长。标准计算：白天净积累 − 夜晚呼吸消耗。
计算公式
白天净积累=净光合速率 × 光照时数
夜晚消耗=呼吸速率 × 暗处时数
一昼夜净积累=白天净积累 − 夜晚消耗 ✓

⚠️ 单位统一很重要：速率通常用 μmol·m⁻²·h⁻¹ 或 mg·h⁻¹，计算前确认单位一致。净积累 > 0 植物能生长；< 0 有机物净亏损；= 0 维持不变。

暗处理实验 黑暗处理一段时间后转到光照，光合速率的变化分析 9/10
EC-422 · 子问题 (1)(2)
核心信号：题目描述"植物先置于黑暗中若干小时，再给予光照"，要求分析ATP、NADPH、C3、C5等物质在转换过程中的变化趋势，或分析光合速率的变化原因。
暗转光的物质变化链
黑暗中→C5消耗（暗反应持续），C5↓；C3积累↑
光照开始瞬间→光反应产生ATP+NADPH→C3被还原
短暂后→C3↓，C5↑（卡尔文循环加速），光合速率恢复 ✓

常考变式：遮光后再开光 vs 正常光照。遮光期间C5减少（呼吸继续但光反应停止），开光后需要一定时间"预热"，光合速率不会立刻达到最大值。
