三级轴系 · 核心调节逻辑
下丘脑（第一级）
分泌促激素释放激素（如 TRH），作用于垂体，启动轴系
垂体（第二级）
分泌促激素（如 TSH），作用于靶腺，放大信号
靶腺（第三级）
分泌靶激素（如 T₃/T₄），作用于全身靶细胞，执行效应
负反馈（调控核心）
靶激素升高 → 抑制下丘脑和垂体 → 减少促激素分泌 → 靶激素回落，维持稳态
高频陷阱（5 个）
切除靶腺后：靶激素↓ → 负反馈解除 → 下丘脑和垂体分泌增加，不是减少
注射大量靶激素后：负反馈增强 → 下丘脑和垂体受抑，促激素↓，靶腺萎缩
切除垂体后：靶激素↓（垂体促激素消失），下丘脑的促释放激素↑（失去垂体负反馈），两者方向相反
激素具有微量高效性——甲亢时 T₃/T₄ 升高幅度不大，但可造成明显临床症状
促激素 vs 促激素释放激素——两者靶器官不同，混淆后整条推理链错误
甲状腺轴 甲亢患者的激素浓度变化分析 9/10
EC-201 · 子问题 (2)(3)
核心信号：甲亢患者甲状腺激素（T₃/T₄）持续偏高 → 负反馈持续增强 → 垂体 TSH 分泌受抑→ TSH 浓度降低；下丘脑 TRH 同样受抑，浓度也降低。三级轴系的负反馈推理范本。
推理信号链
甲状腺激素↑↑ → 负反馈增强 → 垂体 TSH↓
甲状腺激素↑↑ → 抑制下丘脑 → TRH↓ → 轴系整体被抑制 ✓

选题理由：甲亢是三级轴系负反馈的最高频考点，正反方向均需掌握（甲减时方向完全相反）。

轴系结构分析 下丘脑—垂体—肾上腺皮质轴识别 10/10
EC-178 · 子问题 (1)(2)
核心信号：题目给出下丘脑—垂体—肾上腺皮质轴图示，要求识别各级分泌的激素及作用靶器官，同时判断糖皮质激素升高时各级的变化方向。是三级轴系结构识别的标准母题。
三级轴系结构
下丘脑 → CRH → 垂体 → ACTH → 肾上腺皮质 → 糖皮质激素
糖皮质激素↑ → 负反馈至下丘脑 + 垂体 → CRH↓，ACTH↓ ✓

与甲状腺轴结构完全平行——掌握一套轴系逻辑即可迁移到任意一条轴。

负反馈推理 甲减患者长期服用甲状腺激素后的激素变化 9/10
EC-204 · 子问题 (3)
核心信号：外源注射甲状腺激素 → 血液中甲状腺激素↑ → 负反馈增强 → 垂体 TSH 分泌受抑↓ → 靶腺（甲状腺）刺激减少 → 甲状腺萎缩（长期）。注意：外源激素替代的效果等同于靶腺功能亢进，方向与甲亢一致。
推理信号链
外源 T₃/T₄↑ → 负反馈↑ → TSH↓，TRH↓
TSH↓（长期） → 甲状腺刺激减少 → 甲状腺萎缩 ✓

⚠️ 与"甲减（内源性 T₄↓）时 TSH↑"方向完全相反——判断方向前先确认是内源还是外源变化。

切除 / 注射实验 切除垂体后各激素浓度的预测 10/10
EC-183 · 子问题 (4)(5)
核心信号：切除垂体 → 垂体促激素（TSH/ACTH 等）归零 → 靶腺失去促激素刺激 → 靶激素↓↓；同时垂体对下丘脑的负反馈消失 → 下丘脑促释放激素↑↑。两个方向相反，是本模型最高难度陷阱。
切除垂体后的双向变化
垂体切除 → TSH = 0 → 靶激素↓↓
垂体切除 → 垂体对下丘脑负反馈消失 → 下丘脑 TRH↑↑ ✓

选题理由：同一切除操作导致两个方向相反的激素变化，是分级调节的经典高难度题型。
