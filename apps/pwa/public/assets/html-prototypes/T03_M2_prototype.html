路径 A · 花粉不育
配子约束·重算种类与比例
题干声明"含X花粉不育" → 删除受约束雄配子种类 → 母本配子不受约束 → 用可育雄配子×完整母本配子计算合子比例
路径 B · 合子致死
致死型·反推缺失基因型
F2某表型比例为0或缺失 → 正常棋盘写出9种基因型 → 标记致死型（取并集）→ 删除致死格 → 归一化存活比例
路径 C · 连锁互换
配子比例→互换率
测交后代"两大两小" → 大数字=亲本型，小数字=重组型 → 重组率=小数字和/总数 → 互换率=重组率×2（每次互换只产生一半重组型）
高频易错
把花粉不育当成全部配子都不育——"花粉=雄配子"，只约束父本。母本卵细胞不受限，用完整8种母本配子计算。
致死型判断时忘记独立遗传前提——先确认两对基因位于不同染色体，再用9:3:3:1模板定位缺失比例。
混淆重组率与初级卵母细胞互换率——重组率=重组型配子/总配子；互换率=重组率×2（两步分开写，不跳步）。
F2推算时忘记果蝇雄性不互换——雄配子只有亲本型两种（各占50%），F2用4雌×2雄棋盘法。
路径A 母题 花粉不育·配子种类重算·AaBbCc三对基因 ★ 9/10
EC-478 · 雌雄异株植物花粉不育·三对等位基因AaBbCc · 全部子问题
核心考查：含a基因的花粉不育 → 可育雄配子从8种→4种 → 母本配子不变（仍8种）→ 4×8=32种合子按表型归类计算
四步解题逻辑
父本AaBbCc正常8种雄配子→ 删除含a的4种（aBC/aBc/abC/abc）
可育雄配子只剩ABC/ABc/AbC/Abc× 母本8种卵细胞（不变）→ 32种合子归类
关键认知："花粉不育"只约束雄配子——母方配子和雌蕊不受约束。这是M2最高频错误的源头。
三对独立遗传基因配子约束后，B/b×C/c部分仍用乘法原理正常组合。
应用：删除"含a"配子时不需判断B/b、C/c情况，只要带a一律删除。
路径B 母题 双基因致死·F2比例异常·A或b纯合致死 ★ 10/10
EC-481 · 黄色短尾致死·基因A或b纯合致死·两对独立遗传 · 全部子问题
核心考查：两个致死条件叠加（AA致死+bb致死）→ 先各自标记再取并集 → 存活基因型只剩AaB_和aaB_ → 归一化比例
四步解题逻辑
AaBb×AaBb → 9种基因型→ 标记AA（3格）→ 标记bb（3格，取并集）
存活：AaB_ + aaB_→ 比例=2:1（归一化后）
关键认知：两个致死条件叠加时，先各自独立标记再取并集——重叠部分（如AAbb）不重复计算。
致死后比例归一化：分母是存活总数，不再是16。
高频陷阱：看到F2=2:1直接背"显性纯合致死"——本题是双基因致死，结果恰好2:1只是巧合。
路径C 母题 连锁互换·互换率计算·果蝇配子42:8:8:42 ★ 10/10
EC-495 · 果蝇连锁互换·配子比例42比8比8比42·互换率32%·F2表型比71比4比4比21 · 子问题(3)
核心考查：测交后代比例→母方配子比例→分亲本/重组型→重组率→互换率（×2）→F2推算（雄不互换）
完整计算链
测交后代 42:8:8:42→ 母方配子比 AB:Ab:aB:ab = 42:8:8:42
重组率 = (8+8)/100 = 16%→ 互换率 = 16%×2 = 32%
雌配子4种(42:8:8:42)× 雄配子2种(AB:ab=1:1)→ F2 = 71:4:4:21
关键认知："测交后代比例=母方配子比例"是连锁互换计算的万能钥匙。
最大两类=亲本型；最小两类=重组型。
重组率≠互换率——每次交叉互换只产生一半重组型配子，故互换率=重组率×2。
果蝇雄性完全连锁不互换——确认物种后雄配子只写亲本型2种。
从"配子约束识别"到"致死+连锁叠加"到"互换率计算+跨模型联动"，逐步强化配子概率精算能力。
