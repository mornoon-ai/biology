核心调节通路 · 四个节点
光信号输入
视网膜感光细胞接收光照，经视神经传入下丘脑视交叉上核（SCN）
SCN 节律起搏
SCN 是生物钟核心，整合光信号，通过交感神经调控松果体；光照→SCN 抑制松果体
松果体分泌褪黑素
黑暗中褪黑素分泌↑（夜间峰值），光照时抑制分泌；褪黑素是化学信使（激素）
靶器官效应
褪黑素通过血液作用于全身，促进睡眠、调节生殖周期、参与免疫调节
高频陷阱（5 个）
褪黑素分泌受神经调节（交感神经抑制松果体），本身作为激素发挥体液调节——双重属性，不能只选其一
光照→褪黑素减少（光抑制分泌），黑暗→褪黑素增加——方向不能颠倒
SCN 是神经结构（下丘脑核团），松果体是内分泌腺——两者功能不能互换
遮光实验设计：需同时设不遮光对照组，否则无法排除非光因素；遮光材料要确保完全不透光
时差 / 倒班引起节律紊乱——原因是光照时间改变，SCN 重新同步需时间，非褪黑素直接减少
褪黑素分泌规律 24 h 褪黑素分泌曲线解读 9/10
EC-230 · 子问题 (1)(2)
核心信号：曲线显示褪黑素在夜间（约 23:00—凌晨 3:00）出现峰值，白天维持低水平。题目要求解读"峰值时间点""光照对分泌的抑制""遮光后曲线变化"三个层次。是本模型的标准入门母题。
推理信号链
夜间无光照 → SCN 对松果体抑制解除 → 褪黑素↑（夜间峰值）✓
白天光照 → SCN 激活，交感神经抑制松果体 → 褪黑素↓（白天低水平）

选题理由：曲线读图 + 光照因果推理双考点，纯净度高，无需额外背景知识。

SCN 与通路 视交叉上核损毁实验——节律消失机制 10/10
EC-235 · 子问题 (2)(3)
核心信号：损毁 SCN 后，动物昼夜节律消失（活动时间变得随机），褪黑素分泌失去昼夜节律性——证明 SCN 是昼夜节律的"起搏器"。是因果推理型母题，信号链极清晰。
推理信号链
损毁 SCN → 节律起搏器消失 → 无法整合光信号
SCN 信号消失 → 松果体调控紊乱 → 褪黑素节律消失 ✓

⚠️ 关键区分：损毁 SCN ≠ 损毁松果体。SCN 损毁后，松果体本身完好，但失去指令；松果体损毁后，褪黑素直接消失，两者表现不同。

光周期 / 季节 冬季日照短、夜长 → 褪黑素分泌增多 → 生殖抑制 9/10
EC-241 · 子问题 (1)(3)
核心信号：冬季夜长（暗期长）→ 褪黑素分泌时间增长 → 褪黑素总量↑ → 抑制促性腺激素释放 → 生殖器官发育受抑。是光周期—褪黑素—繁殖季节调控的经典模型。
季节性繁殖调控链
冬季夜长 → 暗期延长 → 褪黑素分泌↑（持续时间长）✓
褪黑素↑ → 抑制促性腺激素释放激素 → 性腺发育受抑（非繁殖季）

生态意义：季节性繁殖动物（如绵羊）通过褪黑素感知季节，在食物丰富的春夏季生育后代，提升存活率。

节律紊乱 / 实验设计 模拟时差实验设计与对照组设置 10/10
EC-238 · 子问题 (3)(4)
核心信号：实验设计题——将小鼠的光暗周期突然前移 6 h（模拟跨时区飞行），观察节律重同步（re-entrainment）所需时间及褪黑素分泌峰值时间的偏移。需设置"正常光暗周期组"和"持续黑暗组"两个对照。
实验设计要点
实验组：光暗前移 6 h → SCN 逐渐重新同步 → 褪黑素峰值逐日前移 ✓
对照 1：正常光暗 → 峰值时间不变（基线）
对照 2：持续黑暗 → 自主节律（约 24.x h） → 排除内源节律干扰

实验设计三要素：单一变量（光暗时相）、对照组（正常 + 黑暗两种）、可测量指标（褪黑素峰值时间 / 活动量）。
