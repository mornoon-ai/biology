高频陷阱（5 个）
常显与伴 X 显混淆——关键证据：患病父亲的女儿是否全部患病
常隐与伴 X 隐混淆——关键证据：有无女性患者 + 父亲携带情况
条件概率漏算——"已知表现正常"须调整分母，排除患病个体
杂合子判断——"一定是"与"可能是"混用，须逐一验证能否为纯合子
X 连锁不完全显性——杂合女性可轻度发病，易被误判为常隐
常染色体隐性 镰状细胞贫血系谱判型 9/10
EC-162 · 子问题 (6)(7)
核心信号：单家系同时出现"无中生有"（正常父母生患病孩子）+ "患病女性的父亲表现正常"→ 两个信号共同锁定常染色体隐性，无歧义。
推理信号链
Ⅱ-1 正常×Ⅱ-2 正常 →Ⅲ-1 患病 →无中生有 →排除显性
患病女性 Ⅲ-2 →父亲 Ⅱ-1 正常 →排除伴 X 隐 →常染色体隐性 ✓

选题理由：两个信号同时出现，判型无歧义，是 M1 常隐的标准母题。

常染色体显性 EOFAD 系谱判型 10/10
EC-130 · 子问题 (1)
核心信号："Ⅱ-1 不携带 EOFAD 致病基因"→ 排除隐性携带者传递 → 只能是显性 → 再排除伴 X 显（Ⅰ-1 患病男性有正常女儿）→ 常染色体显性。
推理信号链
Ⅱ-1 不携带致病基因 →排除隐性携带传递 →只能是显性
Ⅰ-1 患病男性 →女儿 Ⅱ-2 正常 →排除伴 X 显 →常染色体显性 ✓

信号链极干净，无需背景知识，单一条件即可完成判型。纯净度 10/10 的标杆母题。

伴 X 染色体隐性 卡尔曼综合征系谱推理 10/10
EC-129 · 子问题 (2)
核心信号（三项锁定）：① 无中生有 → 排除显性；② 患者均为男性 → 提示伴 X；③ 父亲不携带致病基因 → 排除常隐 → 确认伴 X 隐性。
三项锁定
① 无中生有→排除显性遗传
② 患者全为男性→提示伴 X 遗传
③ 父亲不携带 →排除常染色体隐性 →伴 X 隐性 ✓

学生先在 (1) 手绘系谱图，再在 (2) 完成判型——主动建构与推理一体。

伴 X 染色体显性 雷特综合征 RTT 判型 9/10
EC-220 · 子问题 (1)(2)
核心信号：(1) 系谱四选一 → (2) "若 Ⅱ-4 只具致病基因，则所有女儿必患病"——父亲只含致病 X → 女儿全从父获致病 X → 全部患病。这是区分常显与伴 X 显的教科书级证据。
区分常显 vs 伴 X 显的关键推理
父亲基因型 X^A Y →所有女儿必获 X^A →全部患病 →伴 X 显性 ✓

伴 X 显性是高考低频但常混淆考点，EC-220 提供了区分常显与伴 X 显的标准信号。
