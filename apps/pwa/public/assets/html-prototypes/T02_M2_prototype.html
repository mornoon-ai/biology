否决信号 1
跨膜次数计算
跨膜次数 = 穿过完整生物膜的次数。ER膜进1次 + 出1次 = 2次跨膜。不要把双层磷脂当成两次跨膜。分泌蛋白从合成到胞吐全程在膜包被的腔内运输，从不进入细胞质基质。
否决信号 2
自噬体膜来源
自噬体膜来自内质网（ER），不是全新合成的，也不是来自高尔基体。自噬体 = ER来源的双层膜 包裹待降解物 → 与溶酶体融合 → 水解酶降解 → 小分子回收到胞质。
两条膜流路径
路径 1 · 分泌路径
核糖体 → ER → 高尔基体 → 胞吐
核糖体（信号肽引导）→ ER腔（折叠/初加工）→ COPII囊泡 → 高尔基体（修饰/TGN分拣）→ 分泌囊泡 → 细胞膜 → 胞吐。TGN分叉：组成型直接分泌；可调节型储存于分泌囊泡，受信号刺激后释放。
路径 2 · 自噬路径
自噬体膜（来自ER）包裹 → 溶酶体融合 → 降解回收
ER来源的膜 → 双层膜包裹待降解物（受损细胞器/聚集蛋白）→ 自噬体 → 与溶酶体融合 → 自噬溶酶体（水解酶 pH 4.5~5.0）→ 大分子降解为小分子（氨基酸/核苷酸）→ 回收到胞质再利用。
高频易错
跨膜次数多算：一层生物膜 = 一次跨膜（双层磷脂 = 一层膜）。ER进 + ER出 = 2次，不是4次。
自噬体膜来源错误：自噬体膜来自内质网，不是从头合成，不是来自高尔基体。
溶酶体来源混淆：溶酶体由高尔基体出芽形成（高尔基体加工水解酶并包装）。溶酶体 ≠ 自噬体；溶酶体是提供水解酶的一方。
忘记TGN分拣：组成型分泌（如抗体）不需要刺激信号持续分泌；可调节型分泌（如神经递质）储存在囊泡中，受信号触发才释放——两者在TGN处分叉。

EC-013 · (1)(2)(3) · 乳糜微粒形成与脂肪胆固醇转运路径
核心考查：①肠上皮细胞合成乳糜微粒的完整路径 ②分泌蛋白跨膜次数计算 ③COPII囊泡介导的顺向运输方向（ER→高尔基体）
分泌蛋白合成路径
核糖体（信号肽引导）→ ER腔（进1次膜）→ COPII囊泡（出ER膜1次）
高尔基体（修饰/TGN分拣）→ 分泌囊泡→ 细胞膜融合→胞吐
跨膜计数关键：信号肽引导核糖体附着ER→新生肽穿过ER膜进入ER腔（跨1次）；COPII囊泡出芽带走蛋白→穿出ER膜（跨1次）。之后全程在囊泡腔内运输，不再跨膜。总计 = 2次。

乳糜微粒特点：脂蛋白颗粒（脂质+蛋白质），在ER腔内组装。从合成到分泌全程不进入细胞质基质。分泌至淋巴管（胞吐方式），体现生物膜流动性。
路径2 母题 细胞自噬机制·自噬体膜来源·溶酶体融合与降解产物去向 ★ 9/10
EC-002 · (1)(2)(3) · 细胞自噬机制与溶酶体功能分析
核心考查：①自噬体膜来源（ER，非高尔基体）②自噬体-溶酶体融合形成自噬溶酶体 ③降解产物（氨基酸/核苷酸等）回收到胞质再利用
自噬路径
ER来源的膜→ 双层膜包裹待降解物→ 自噬体
与溶酶体融合→ 自噬溶酶体（水解酶激活）→ 小分子回收胞质
自噬体膜来源：内质网（ER）提供双层膜——注意不是高尔基体，也不是全新合成。

降解产物去向：水解酶将大分子（蛋白质/核酸/脂质）降解为小分子（氨基酸/核苷酸/脂肪酸）→ 穿过溶酶体膜 → 回到胞质 → 被细胞再利用（循环再生）。

溶酶体酸性环境：溶酶体内 pH 4.5~5.0，H⁺由H⁺-ATPase主动泵入维持。酸性环境是水解酶发挥活性的必要条件（若pH升高，水解酶活性下降）。
从"分泌路径完整追踪"到"路径分叉与选择性自噬"到"pH调节机制"，逐步深化内膜系统的动态协作理解。
