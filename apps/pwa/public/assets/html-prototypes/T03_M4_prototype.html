三条技术路线
路径 A · 直接电泳
条带位置→等位基因→基因型
题面给出电泳图 → 相同位置=相同等位基因 → 1条带=纯合(两个相同等位基因)，2条带=杂合 → 对照已知基因型个体的条带位置逐一判读
路径 B · RFLP
酶切位点改变→片段长度→条带差异
限制酶识别特定序列 → 突变创建新位点(多切一刀→条带增多)或破坏位点(少切一刀→条带减少) → 片段长度组合反映等位基因状态 → 连锁分析定位致病基因
路径 C · STR/SSR
重复次数不同→PCR产物长度→分型
短串联重复序列(STR)在不同个体间重复次数不同 → PCR扩增 → 电泳显示不同长度条带 → 与致病基因紧密连锁→间接追踪致病基因传递 → 亲子鉴定/连锁诊断
高频易错
认为三条带=三倍体——RFLP杂合子酶切后可出现3条带（2个等位基因各产生不同数量片段），条带数≠染色体组数。
条带变多→突变破坏酶切位点（方向反了）——突变创建新位点→多切→条带增加；突变破坏位点→少切→条带减少。
把RFLP标记当作致病基因——标记是与致病基因连锁的中性路标，本身不致病，诊断价值来自"连锁"不来自"功能"。
STR分型匹配→"确认"亲子——只能说"不排除"；确认需多位点联合，随机匹配概率极低但非零。
路径A 母题 镰状细胞贫血·电泳条带→HbA/HbS等位基因 ★ 10/10
EC-162 · 疟原虫/镰状细胞贫血HbS基因频率进化与系谱分析综合 · (7)
核心考查：表2给出1~4号个体Hb电泳条带 → 条带1含哪个基因(HbA/HbS) → 不依赖背景知识，纯条带→等位基因映射
四步解题
定位标准条带→ 已知纯合个体条带位置=该等位基因→ 对照判断未知个体
1条带=纯合 / 2条带=杂合→ 写出基因型
关键认知：HbA与HbS因氨基酸替换→电荷不同→电泳迁移率不同→跑到不同位置。
条带高度=分子迁移速率，不同等位基因→不同位置。纯合子2个相同等位基因→同一位置叠加→1条带；杂合子→2条带。
路径B 母题 RFLP双酶切+连锁→常隐致病基因定位 ★ 9/10
EC-498 · RFLP分子标记·SphI_HindIII酶切·常隐系谱连锁分析·致病基因定位 · (2)(3)
核心考查：SphI+HindIII两标记完全连锁 → 从酶切片段推断父/母本与致病基因连锁的RFLP长度 → 再推断子代基因型与交换概率
四步解题
系谱判遗传方式（常隐）→ 读父本SphI标记（7.0kb）
从患者子代倒推母本标记连锁→ 确定致病基因连锁的RFLP组合
关键认知：RFLP标记本身不致病，是与致病基因物理连锁的中性标记。子代只有某种标记组合→亲本对应染色体上标记必为该连锁方式（倒推法）。
路径C 母题 STR标记+伴X连锁→血友病间接诊断 ★ 8/10
EC-109 · STR短串联重复序列亲子鉴定与葡萄胎染色体来源分析及伴X遗传连锁判断 · (3)
核心考查：血友病(伴X隐)+DX1205(X染色体STR)完全连锁 → 追踪STR等位基因传递=追踪致病基因传递 → 可间接诊断未出生个体
四步解题
确认血友病=伴X隐→ 读DX1205分型（X染色体STR）
追踪患者X染色体上的STR等位基因→ 间接诊断=高概率推断（非100%）
性别差异：男性XY→X染色体STR只有一条带；女性XX→可能一或两条带。
连锁诊断本质：因为STR与致病基因物理位置接近→很少被互换分开→STR传递≈致病基因传递，但不是100%确定。
从"纯条带判读"到"系谱+RFLP串联"到"系谱·条带·概率三合一闭环"，逐步强化分子证据判读能力。
