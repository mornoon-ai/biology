核心概念 · 四个分析维度
有氧呼吸方程式
C₆H₁₂O₆ + 6H₂O + 6O₂ → 6CO₂ + 12H₂O + 能量；CO₂释放量 = O₂消耗量，RQ=1；产生大量ATP（约38mol）
无氧呼吸方程式
植物/酵母：C₆H₁₂O₆ → 2C₂H₅OH + 2CO₂；动物/乳酸菌：C₆H₁₂O₆ → 2C₃H₆O₃；酒精途径有CO₂释放，乳酸途径无气体变化
混合呼吸产物比例计算
测量CO₂释放量（总）和O₂消耗量：有氧部分CO₂=O₂消耗量；无氧CO₂=总CO₂−有氧CO₂；据此推算两种呼吸消耗葡萄糖的比
贮藏与保鲜应用
低O₂（但>2%）+低温→降低呼吸速率→减少有机物消耗→延长保质期；完全无O₂→无氧呼吸产生酒精→反而有害
高频陷阱（5 个）
无氧呼吸产酒精的CO₂释放量不等于O₂消耗量（无氧呼吸不消耗O₂）——CO₂释放 > O₂消耗，说明存在无氧呼吸（酒精途径）
乳酸途径（动物、乳酸菌）无CO₂释放、无气体体积变化——密封容器中气体不变不代表不呼吸，可能是纯乳酸发酵
有氧呼吸方程式中，水是反应物（参与第二阶段，6H₂O进入线粒体基质），CO₂由丙酮酸脱羧产生——不能说"有氧呼吸不需要水"
贮藏粮食/蔬菜时应用低O₂不是无O₂——完全无氧会产生大量酒精，对细胞有毒；适当低O₂（2-5%）才是最优选择
ATP产量：有氧呼吸远多于无氧呼吸——1 mol葡萄糖有氧≈38 mol ATP，无氧（酒精途径）只产生2 mol ATP——不能说"无氧产能更多"
有氧呼吸 有氧呼吸三个阶段的场所、物质变化与ATP产量 10/10
EC-460 · 子问题 (1)(2)
核心信号：题目要求写出有氧呼吸的总方程式，或填写三个阶段的场所、底物、产物；或分析某阶段产物中H和O的来源。标准母题：三阶段逐一分析。
有氧呼吸三阶段速记
第一阶段（细胞质基质）：葡萄糖→丙酮酸+[H]+少量ATP
第二阶段（线粒体基质）：丙酮酸+H₂O→CO₂+[H]+少量ATP
第三阶段（线粒体内膜）：[H]+O₂→H₂O+大量ATP ✓

来源辨析：CO₂中的O来自葡萄糖和水（第二阶段）；H₂O中的H来自葡萄糖和水（[H]来自第一、二阶段），O来自O₂（第三阶段）。这是常考的同位素标记追踪题目的分析基础。

无氧呼吸 植物无氧呼吸（酒精途径）与动物无氧呼吸（乳酸途径）的产物区别及判断 10/10
EC-464 · 子问题 (1)(2)
核心信号：题目要求区分植物/酵母菌的无氧呼吸和动物/乳酸菌的无氧呼吸，或在密封容器实验中，根据CO₂和O₂的变化量判断呼吸方式。
两种无氧呼吸对比
酒精途径（植物/酵母）：葡萄糖→酒精+CO₂（有CO₂释放）
乳酸途径（动物/乳酸菌）：葡萄糖→乳酸（无气体变化）✓
密封容器判断逻辑
CO₂释放 = O₂消耗→纯有氧呼吸（RQ=1）
CO₂释放 > O₂消耗→有氧+无氧（酒精途径）混合 ✓
气体总量不变→纯无氧（乳酸）或有氧+乳酸混合 ✓

⚠️ 用NaOH（吸收CO₂）和清水（不吸收气体）分别测量气体体积变化，可区分O₂消耗量和CO₂释放量，是经典实验设计考点。

混合比例计算 由O₂消耗量和CO₂释放量推算有氧/无氧呼吸消耗葡萄糖的比例 10/10
EC-468 · 子问题 (2)(3)
核心信号：题目给出"O₂消耗量为a mol，CO₂释放量为b mol（b>a）"，要求计算有氧和无氧呼吸各消耗多少葡萄糖。标准方程组解法。
计算方程组
有氧部分：CO₂释放 = O₂消耗 = a mol
无氧部分CO₂=总CO₂(b) − 有氧CO₂(a) = (b-a) mol
无氧葡萄糖=(b-a)/2 mol（每mol葡萄糖产2mol CO₂）✓
有氧葡萄糖=a/6 mol（每mol葡萄糖消耗6mol O₂）✓

注意：此公式适用于植物/酵母（无氧产酒精+CO₂）。若为动物无氧（乳酸途径），CO₂只来自有氧部分，无氧部分不释放CO₂，计算逻辑不同。

贮藏保鲜应用 低O₂贮藏蔬菜水果的最优条件分析与无氧呼吸危害评估 9/10
EC-472 · 子问题 (1)(3)
核心信号：题目给出不同O₂浓度下蔬菜/水果的呼吸速率曲线或有机物消耗量，要求判断最适O₂浓度，并解释为何不能完全密封。
最优贮藏O₂浓度分析
完全无O₂（密封）→纯无氧呼吸→酒精积累→细胞受损
O₂过高（21%）→纯有氧呼吸→消耗大量有机物
最优O₂约2~5%→有氧+少量无氧→总呼吸最低→最省有机物 ✓

低温同时配合低O₂效果最佳：低温降低酶活性→减慢呼吸速率；低O₂抑制有氧呼吸→两者协同减少有机物消耗。但要避免完全无O₂造成无氧呼吸酒精积累。
