核心识别信号
题型触发词
细胞骨架异常 / 微管蛋白突变
关键结构
微管 · 囊泡 · 纺锤体 · 细胞膜
下游后果

考查逻辑
结构变化 → 功能丧失 → 因果链推演
三条子专题
子专题 1
微管受损 → 囊泡运输瘫痪
TUB / OTOF 基因突变 → 细胞骨架异常 → 分泌囊泡无法沿微管轨道运输 → 靶细胞功能衰竭（不育 / 耳聋）
子专题 2
微管受损 → 纺锤体 / 分裂异常
TUBB8 / TRIP13 突变 → 纺锤体不能正常组装或无法完成双极化 → 减数分裂停滞 → 卵子成熟障碍
子专题 3
膜结构变化 → 凋亡小体形成
凋亡信号 → 细胞皱缩 → 细胞膜内陷（体现流动性）→ 形成凋亡小体 → 被吞噬 → 溶酶体水解降解
高频易错
微管受损 ≠ DNA复制障碍。突变只影响细胞骨架，DNA复制功能正常——突变细胞能进入分裂，只是纺锤体缺失导致停滞。
囊泡无法运输 → Ca²⁺浓度升高而非降低。囊泡积压在细胞内，Ca²⁺无法正常释放，内部浓度反而偏高。
凋亡小体由细胞膜内陷断裂形成，体现膜的流动性；但溶酶体降解不体现流动性，而体现酶的催化性。
纺锤体"形成失败"（子专题2母题）vs"双极化失败"（进阶变式）是不同阶段缺陷，需根据题目信息区分。

子专题2母题 TUBB8突变→纺锤体缺失→减Ⅰ停滞 ★ 9/10
EC-166 · (1)(2) · 2026-05-20
核心因果链：TUBB8突变 → 纺锤体相关蛋白合成受阻 → 纺锤体缺失（荧光标记无绿色信号）→ 卵母细胞停留在减Ⅰ中期不能继续分裂 → 第一极体产生量下降
荧光实验关键
正常卵母细胞：绿色+蓝色荧光vs 突变卵母细胞：只有蓝色荧光（无纺锤体）
DNA已复制（蓝色+）但 细胞不能分裂（纺锤体-）→选ACD
(3)题：致病TUBB8基因来自父亲 → 说明该基因在精子形成过程中不表达（B正确），而非影响精子成熟或位于Y染色体上。
子专题3母题 凋亡蛋白→细胞膜内陷→凋亡小体形成→溶酶体降解 ★ 8/10
EC-494 · (3)(4) · 2026-05-24
核心因果链：促凋亡蛋白激活 → 细胞皱缩 → 细胞膜向内凹陷形成小囊泡 → 断裂形成凋亡小体 → 被相邻细胞或吞噬细胞吞噬 → 溶酶体水解酶彻底降解
两个考查点
(3) 凋亡过程顺序| (4) 膜内陷→体现流动性
区分：内陷形成凋亡小体体现膜的流动性；溶酶体酶水解体现酶的专一性/高效性，不体现流动性。不释放胞内物质 → 不引起炎症反应（区别于坏死）。
