两条子专题
子专题 1
基因组印记 — DNA甲基化致单等位基因表达
亲源印记→一方等位基因启动子CpG岛甲基化→转录沉默→只有非甲基化等位基因表达（Igf-2：父源表达，母源沉默）。配子形成时印记擦除重建。印记异常或突变双重打击→疾病（天使综合征）
子专题 2
甲基化结合蛋白异常 → 转录调控丧失
MeCP2蛋白→结合甲基化CpG→招募HDAC→染色质压缩→转录抑制。MECP2突变→无法结合→靶基因（BDNF）异常高表达→神经突触异常→雷特综合征
高频易错
表观遗传≠基因突变。甲基化修饰不改变碱基序列（DNA序列不变），但可影响基因表达。考试中"碱基序列改变/未改变"是判断依据。
Igf-2印记规律：只表达来自父本的等位基因（父源非甲基化→表达；母源甲基化→沉默）。杂合子Aa中，表达父源基因——若父源为A则正常，若父源为a则矮小。
印记重建：配子形成过程中印记擦除并按性别重建。雌性配子重建为甲基化（沉默），雄性配子重建为去甲基化（表达）。
天使综合征双重打击：父本UBE3A甲基化沉默（正常印记）+ 母本UBE3A基因突变 → 两个等位基因均无功能 → 患病。
MECP2突变：MECP2蛋白失功能→无法抑制BDNF表达→BDNF过高（不是过低）→神经发育异常。注意方向：抑制因子失活→靶基因上调。
组蛋白甲基化（H3K4met）修饰不改变DNA碱基序列，但属于表观遗传修饰，影响基因转录（转录水平调控）。
子专题1 母题 Igf-2基因印记·DNA甲基化·印记重建·单亲胚胎异常 ★ 10/10
EC-144 · (1)(2)(3)(4)(5)(6)(7) · 2026-05-19
核心因果链：Igf-2基因→母源等位基因启动子甲基化→转录沉默→只有来自父本的Igf-2等位基因表达。配子形成时印记按性别重建：雌配子→甲基化（沉默），雄配子→去甲基化（表达）。单亲胚胎→两等位基因同时沉默或同时表达→发育异常
印记规律与表型推断
AA父源A表达→ 正常
Aa只表达父源→ 父源A→正常 / 父源a→矮小
aa只表达父源a→ 矮小
考点(1)：AA→父源A→正常；Aa→只表达父源（若父→A则正常）；aa→矮小。答案：正常 正常 矮小（选A）。

考点(2)印记交叉：甲组F1雄鼠Aa → 精子A:a=1:1（精子形成时去甲基化，均可表达） → F2中来自父本精子A→正常，来自父本精子a→矮小 → 正常:矮小=1:1。

考点(3)甲基化特点：A可遗传（代间传递）、C与细胞质有关（雌雄配子环境不同导致重建方向不同）、D影响基因表达。B错（不改变碱基序列，非基因突变）。选ACD。

考点(4)：碱基序列未改变；调控在转录水平（甲基化阻止RNA聚合酶结合启动子）。

考点(6)实验设计：验证雌配子A基因印记重建→选①正常纯合雌鼠(AA)×⑤矮小纯合雄鼠(aa)→子代Aa；若雌配子A甲基化沉默→只表达来自父本的a→全部矮小（验证雌方A被甲基化沉默）。

考点(7)单亲胚胎：全父本→两个等位基因均去甲基化→均表达Igf-2→过量→发育异常；全母本→均甲基化→均沉默→无Igf-2→发育异常。
子专题2 母题 雷特综合征MECP2·BDNF转录调控·组蛋白甲基化 ★ 9/10
EC-094 · (1)(2)(3)(4)(5)(6)(7) · 2026-05-18
核心因果链：正常MECP2→结合甲基化CpG→招募HDAC→染色质压缩→BDNF转录抑制。MECP2突变→无法结合→HDAC不招募→BDNF过度转录→神经突触异常→RTT。治疗思路：恢复MECP2功能 / 提高BDNF到适当水平（BHBA→H3K4met上升→BDNF表达促进）
BHBA提高BDNF的分子通路
BHBA→ Ca²⁺通道开放↑→ CaMKⅡ磷酸化↑→ H3K4met↑→ BDNF转录↑
考点(1)精子形成中含致病基因细胞：该患者只有一条染色体上MECP2突变（杂合子）。减数分裂前S期→两条染色体均复制。初级精母细胞②含致病基因（复制后每个细胞含）；次级精母细胞③可能含2个致病基因（同源染色体不分离/互换）。精原细胞①一定含（有丝分裂保留）；②初级精母细胞一定含。可能含两个致病基因的有②⑤（减一不分离）。

考点(2)H3K4met：组蛋白修饰影响DNA缠绕状态→影响转录（C对）；不改变DNA碱基序列（A/B错）；不在翻译水平（D错）。选C。

考点(3)RTT患者BDNF含量：MECP2突变→无法抑制BDNF→BDNF含量减少（注意：题干说MECP2正常时抑制BDNF，但RTT中神经细胞BDNF下调是另一机制——需结合题目具体图3）。按题意MECP2突变→BDNF减少→神经细胞程序性死亡增加。

考点(5)错误叙述：MECP2基因≠蛋白质（不与双缩脲反应，A错）；BDNF合成→mRNA种类/数量变化但DNA种类不变（B错）；BDNF 119个氨基酸→最少357个碱基（D错：357个碱基=119×3密码子，但基因还有非编码区，"至少"应是119×3×2=714个碱基对）。选ABD。
从"单一基因印记机制"到"印记+突变双重打击"到"甲基化结合蛋白调控通路"，逐步深化表观遗传的分子理解。
