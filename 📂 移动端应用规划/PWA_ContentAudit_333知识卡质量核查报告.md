# PWA 333 张知识卡质量核查报告

日期：2026-06-12

## 一、核查目标

本阶段开始进入内容质量修订，不再只处理界面和跳转问题。

重要修正：知识卡不是从讲稿里摘句子生成的卡片。知识卡应当是母题卡的拆解结果，是构建母题卡、训练母题动作、积累母题框架的基础颗粒。讲稿只能作为讲解辅助，不能作为知识卡的源头。

本轮核查对象为 `knowledge_cards.json` 中的 333 张知识卡。核查标准是：

1. 卡片内容必须与卡片名称、类型、章节功能相符合。
2. 卡片必须能追溯到母题综合卡、图片记忆卡、审题触发卡、解题程序卡、机制链卡、得分表达卡、母题子训练或限时变式题。
3. 正面必须能提出一个明确训练任务，不能只是讲稿片段。
4. 背面必须能帮助学生调用母题框架解决实际题目，不能只是重复正面。
5. 信号卡、规则卡、陷阱卡、代表题卡必须各自承担清楚功能。
6. 卡片应能沉淀为可复测、可积累的学习材料。

## 三、主要问题

1. 来源链路错误  
   当前大量卡片来自 `Txx_Mx_script.md`。这是最根本的问题：讲稿是教学叙述，母题卡才是系统骨架。知识卡必须回到母题综合卡、图片记忆卡、触发卡、程序卡、机制链、得分句、子训练、变式题。

2. 正反面重复或答案未加工  
   很多卡片正面和背面只是同一段内容的改写，学生看答案后不能获得新的判断方法。

3. 代表题卡没有形成真实迁移任务  
   当前 56 张代表题卡全部命中风险。问题集中在：只说明“第一道母题/第二道母题”，没有给出小题面、对应母题框架、迁移点和作答动作。

4. 规则卡没有形成规则  
   92 张规则卡全部命中风险。规则卡应给出“先看什么、再排什么、最后怎么写”的判断流程，而不是保留讲稿叙述。

5. 信号卡没有明确识别线索  
   信号卡应告诉学生“看到什么就知道进入这个母题”，例如题干关键词、图像特征、问法、限制条件。目前多数信号卡还没有做到。

6. 陷阱卡没有明确错误动作  
   陷阱卡应该指出学生会错在哪里，以及如何避开。现在部分陷阱卡只是提示一句话，不足以用于复测。

7. 卡片类型与章节功能混淆  
   有些卡片位于“拆题/判断/框架”等章节，但类型和内容没有形成明确对应，导致筛选和复测时学生不知道自己在练什么。

## 四、正确来源链

每一张知识卡都必须优先追溯到下面这些“祖宗”材料：

| 来源层级 | 作用 |
| --- | --- |
| 母题综合卡 | 定义母题名称、触发信号、母题精讲、变式阶梯、易错点 |
| 图片记忆卡 | 提供学生第一眼看到的母题结构总览 |
| 审题触发卡 | 生成信号卡 |
| 解题程序卡 | 生成规则卡、流程卡 |
| 机制链卡 | 生成因果链、解释链、推理链卡 |
| 得分表达卡 | 生成标准表述、短答案卡 |
| 母题子训练 | 生成最小动作训练卡 |
| 限时变式题 | 生成代表题卡与迁移卡 |

讲稿的位置应降级为：解释材料、补充口语化说明、音频对齐材料。讲稿不能直接决定知识卡正反面。

## 五、各类卡片重写标准

| 卡片类型 | 应承担的功能 | 合格标准 |
| --- | --- | --- |
| 信号卡 | 识别题型入口 | 正面问“看到什么进入本母题”；背面列出关键词、图像信号、问法和限制条件 |
| 规则卡 | 固化判断流程 | 背面必须有步骤，例如“先判显隐，再判常/X，最后处理概率或特殊证据” |
| 陷阱卡 | 防止典型误判 | 正面呈现易错动作；背面说明为什么错、正确替代动作是什么 |
| 代表题卡 | 训练迁移应用 | 正面给出微型题面；背面对应母题框架、迁移点和得分表达 |
| 回忆卡 | 保留核心口诀或结论 | 只能用于钩子/收束等轻量内容，也必须能独立回答一个明确问题 |

## 六、建议返工顺序

第一优先级：重建来源链  
原因：先把每张卡的来源从讲稿改回母题卡体系，否则后面改写会继续跑偏。

第二优先级：56 张代表题卡  
原因：它们最直接决定学生能否把母题框架迁移到真实题目中，也是当前全部命中风险的一类。

第三优先级：92 张规则卡  
原因：规则卡是母题框架的骨架，决定学生是否能稳定复现解题流程。

第四优先级：58 张信号卡与 76 张陷阱卡  
原因：信号卡解决“进哪类题”，陷阱卡解决“为什么会错”。这两类应在规则卡之后统一校准。

第五优先级：51 张回忆卡  
原因：回忆卡数量较少，可在前几类修订后统一清理空泛卡和重复卡。

## 八、阶段结论

本阶段已经建立 333 张卡片的内容质量审计基线，并修正审计原则：知识卡必须从母题卡体系拆解，不从讲稿抽取。

建议先以 T03_M1 为样板，从 `母题综合卡 -> 审题触发卡 -> 解题程序卡 -> 母题子训练 -> 限时变式题` 重新生成一组可验收的高质量卡片模板，再批量扩展到所有 29 个母题。验收标准是：学生只看卡片，不回看讲稿，也能知道“这张卡练什么、怎么答、如何迁移到题目”。
