## 三、样板结构

这 12 张卡覆盖一条完整学习链：

1. 钩子：先判显隐，再判常/X，防止跳步。
2. 信号：识别系谱图、家系描述、问法与题干关键词。
3. 判断：固化五步判型程序和一票否决规则。
4. 框架：统一常隐、常显、X隐、X显四类遗传方式。
5. 拆题：用代表题训练常隐判型和条件概率。
6. 陷阱：纠正“无中生有直接常隐”和“正常子代概率算 1/2”。
7. 收束：形成自检句，并说明如何迁移到电泳、概率、减数分裂综合题。

## 五、后续批量标准

后续每个母题建议保持同样的 12 卡结构：

| 位置 | 功能 |
| --- | --- |
| 钩子 1 张 | 明确本母题最容易跳错的第一动作 |
| 信号 2 张 | 题面入口信号 + 核心判别信号 |
| 判断 2 张 | 主流程规则 + 关键排除规则 |
| 框架 1 张 | 母题总框架 |
| 拆题 2 张 | 代表题迁移 + 得分表达或计算动作 |
| 陷阱 2 张 | 高频错因 + 纠偏动作 |
| 收束 2 张 | 自检句 + 跨母题迁移 |

验收标准：每张卡必须能说清“练什么、怎么做、来自哪个母题源、错了如何纠偏”。
